## Contexto Técnico

### Breaking Change Crítica: InstallmentKind

**Antes (0.18.0):**
```ts
enum InstallmentKind {
  monthly = 'monthly',
  yearly = 'yearly',
  entry = 'entry',
  extra = 'extra',
}
```

**Depois (1.0.0):**
```ts
enum InstallmentKind {
  entry = 'entry',
  regular = 'regular',       // era monthly/yearly recorrente
  balloon = 'balloon',
  key_delivery = 'key_delivery',
  extra = 'extra',
}

enum InstallmentPeriodicity {
  one_shot = 'one_shot',
  monthly = 'monthly',
  bimonthly = 'bimonthly',
  quarterly = 'quarterly',
  semestral = 'semestral',
  yearly = 'yearly',
}
```
